# 풀이 방법

---

1. 메모이제이션을 사용한 방법

   ```js
   let result = Array(100001).fill(-1);

   function solution(n) {
     if (result[n] != -1) {
       return result[n];
     }

     if (n <= 0) {
       return 0;
     }

     if (n <= 2) {
       return 1;
     }

     result[n] = (solution(n - 1) + solution(n - 2)) % 1234567;

     return result[n];
   }
   ```

   - n은 2 이상 100,000 이하인 자연수라는 조건이 있으므로, 100,001개인 배열 result를 생성하여 인덱스마다 -1일씩 값을 준다.
   - n이 0이면 0을 반환, 2 이하이면 1을 반환한다.
   - 그리고 result[n]에 값이 n일 때 해당하는 피보나치 수를 할당한다.
   - 이제 solution 함수가 재귀될 때마다, 배열의 n번째 인덱스 값을 검사하게 된다. result[n]의 값이 -1이 아니라면 result[n]의 값이 할당되어있다는 뜻이므로 계산할 필요 없이 result[n]을 바로 반환한다. -> `메모이제이션`
   - 재귀를 사용한 피보나치 수열의 시간복잡도는 O(2^n)인데 메모이제이션을 사용하면 `O(n)`으로 줄일 수 있다.
   - 하지만 메모이제이션을 사용해도 13, 14번째 테스트 통과가 안된다 🥲

2. 피보나치 수 쉽게 푸는 방법

   ```js
   function solution(n) {
     let a = 0;
     let b = 1;
     let c = a + b;

     for (let i = 2; i < n; i++) {
       a = b;
       b = c;
       c = (a + b) % 1234567;
     }

     return c;
   }
   ```
