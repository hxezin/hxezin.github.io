# 동적 인라인 스타일 설정하기

---

```js
const [isValid, setIsValid] = useState(true);

const formSubmitHandler = (event) => {
  event.preventDefault();
  if (enteredValue.trim().length === 0) {
    setIsValid(false);
    return;
  }
};
```

````js
<label style={% raw %}{{ color: !isValid ? "red" : "black" }}{% endraw%}>Course Goal</label>
```

- 현재 입력 값이 유효한지에 대한 state를 만들고, state 값에 따라 색상을 동적으로 적용 가능하다.
- 문제점: 스타일이 변한 후 입력값을 제대로 넣어도 다시 스타일이 `리셋되지 않는다.` → 유지보수X
- 인라인 스타일은 최우선순위로 적용되어 css 파일을 오버라이드 한다.

<br /><br /><br /><br />

# 동적으로 CSS 클래스 설정하기

---

```css
.form-control.invalid input {
  background-color: red;
  background: #ffd7d7;
}

.form-control.invalid label {
  color: red;
}
````

- className으로 form-control과 invalid를 동시에 갖는 요소만 선택하는 style을 생성한다.

<br />

```js
<div className={`form-control ${!isValid ? "invalid" : ""}`}>
  <label style={% raw %}{{ color: !isValid ? "red" : "black" }}{% endraw %}>Course Goal</label>
  <input type="text" onChange={goalInputChangeHandler} />
</div>
```

- 삼항연산자를 사용해서 state 값에 따라 className을 동적으로 변화시킬 수 있다.
- `템플릿 리터럴` 사용!
- 문제점: 프로젝트가 커질수록 중복되는 className이 나올 수 있다.

<br /><br /><br /><br />

# Styled Components

---

```bash
npm install --save styled-components
```

- Styled Components 패키지 설치 필요

<br />

```js
const button = styled.button`내용`;
```

- 태그드 템플릿 리터럴(Tagged Template Literals)
- button은 style라는 객체의 메서드이고, styled는 styled-components에서 가져온 객체

<br />

```js
import styled from "styled-components";

const Button = styled.button`
  width: 100%;
  font: inherit;
  padding: 0.5rem 1.5rem;
  border: 1px solid #8b005d;
  color: white;
  background: #8b005d;
  box-shadow: 0 0 4px rgba(0, 0, 0, 0.26);
  cursor: pointer;

  @media (min-width: 768px) {
    width: auto;
  }

  &:focus {
    outline: none;
  }

  &:hover,
  &:active {
    background: #ac0e77;
    border-color: #ac0e77;
    box-shadow: 0 0 8px rgba(0, 0, 0, 0.26);
  }
`;
```

- 가상선택자의 경우 `&` 기호를 사용한다.
  → 여기서 &는 button을 참조함

<br />

- 개발자 도구로 Styled Component가 동적으로 생성한 class를 확인할 수 있다. Styled Components가 생성한 모든 클래스는 `고유한 이름`을 갖기 때문에 중복될 일이 없어 다른 컴포넌트에 영향을 주지 않는다.

## Styled Components & 동적 Props

```js
const FormControl = styled.div`
  margin: 0.5rem 0;

  & label {
    font-weight: bold;
    display: block;
    margin-bottom: 0.5rem;
    color: ${(props) => (props.invalid ? "red" : "black")};
  }

  & input {
    display: block;
    width: 100%;
    border: 1px solid ${(props) => (props.invalid ? "red" : "#ccc")};
    background: ${(props) => (props.invalid ? "#ffd7d7" : "transparent")};
    font: inherit;
    line-height: 1.5rem;
    padding: 0 0.25rem;
  }

  & input:focus {
    outline: none;
    background: #fad0ec;
    border-color: #8b005d;
  }

  &.invalid label {
    color: red;
  }
`;
```

- CourseInput 파일에 FormControl 컴포넌트를 하나 더 생성한다. 여기에 CourseInput.css 내용을 넣어준다.

<br />

```js
<form onSubmit={formSubmitHandler}>
  <FormControl invalid={!isValid}>
    <label>Course Goal</label>
    <input type="text" onChange={goalInputChangeHandler} />
  </FormControl>
  <Button type="submit">Add Goal</Button>
</form>
```

- FormControl 컴포넌트를 사용해서 invalid를 props로 보낸다.

<br />

```js
& label {
    font-weight: bold;
    display: block;
    margin-bottom: 0.5rem;
    color: ${(props) => (props.invalid ? "red" : "black")};
  }
```

- 그럼 FormControl 컴포넌트에서 props를 매개변수로 하는 함수를 사용해 텍스트를 반환한다.

## Styled Components & 미디어 쿼리

```css
@media(조건){
	컴포넌트에 적용할 스타일
}
```

- 이런 형식으로 미디어 쿼리를 만들 수 있다.

<br />

```js
const Button = styled.button`
  width: 100%;
  ...

  @media (min-width: 768px) {
    width: auto;
  }
...
```

- 창 너비가 768px보다 작아지면 버튼 너비는 100%가 된다.

# CSS 모듈 사용하기

---

- css 모듈을 사용하기 위해선 css 파일 이름에 `Button.module.css` 처럼 중간에 module을 추가한다.

<br />

```js
import styles from "./Button.module.css";

const Button = (props) => {
  return (
    <button type={props.type} className={styles.button} onClick={props.onClick}>
      {props.children}
    </button>
  );
};
```

- `import style from ‘./Button.module.css’` 형식으로 임포트 한다.
  - styles 대신 다른 이름을 사용해도 상관 없지만 보통 styles / classes를 사용한다.
- className에 문자열 대신 동적인 값을 적용하기 위해 styles를 참조한다. styles는 객체 형식이므로 css파일에서 프로퍼티로 정의되는 모든 클래스를 갖는다.

<br />

![img](/assets/img/blog/220611-2.png)

- 개발자 도구로 class에 “컴포넌트 이름-클래스 이름-고유한 해시값"이 추가된 것을 확인할 수 있다.
- css 모듈 내부에서 수행하는 구축 프로세서는 css 클래스나 css 파일을 가지고 `클래스 이름을 고유한 이름으로 바꾸는 것`이 핵심 작업이다.

<br /><br />

## CSS 모듈을 사용한 동적 스타일

```js
import styles from "./CourseInput.module.css";

<form onSubmit={formSubmitHandler}>
  <div className={`${styles["form-control"]} ${!isValid && styles.invalid}`}>
    <label>Course Goal</label>
    <input type="text" onChange={goalInputChangeHandler} />
  </div>
  <Button type="submit">Add Goal</Button>
</form>;
```

```css
.form-control.invalid input {
  border-color: red;
  background: #ffd7d7;
}

.form-control.invalid label {
  color: red;
}
```

- 전에 했던 것처럼 isValid 값이 유효한지 체크한 후 invalid className을 동적으로 변환할 수 있다.

<br />

```js
@media (min-width: 768px) {
  .button {
    width: auto;
  }
}
```

- css모듈은 선택자가 필요하다. css모듈로 미디어 쿼리를 사용하기 위해서는 선택자를 추가해줘야 한다.
